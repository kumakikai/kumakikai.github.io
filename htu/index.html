How to Use

現在、使い方の詳細は公式サイトへ順次移行中です。

反映が完了するまでの間は、これまで note で公開していた以下の記事をご参照ください。

  • 画面付きでの使い方解説（note）

1. 収支を記録する#

トップ画面で ↑ / ↓ を選び、金額を入力します。
入力は最小限で、迷う要素はありません。

2. 日付を変更する#

過去の日付を選択して記録・修正が可能です。
※未来日は選択できません。

3. タグを付ける#

最大4つまでタグを設定できます。
カテゴリ分けは任意です。

4. 履歴の削除とUndo#

誤って登録した場合でも、削除後すぐに取り消せます。

5. カレンダーで振り返る#

  • 日別
  • 月別
  • カテゴリ別

数字を並べて見ることだけに集中できます。

6. 設定#

  • テーマ（ライト / ダーク）
  • 言語（日本語 / 英語）
  • タグ表示形式

※ バックアップや同期機能は現在提供していません。
